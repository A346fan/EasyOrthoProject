Install instructions:

1. Extract the downloaded file anywhere on your PC (recommended on the same drive the Sim is located)
2. Search for the Folder containing files named like "zOrtho4XP_+00+000"
3. Link those folders into your Custom Scenery directory
4. Start X-Plane and quit it once you're in the main menu
5. Open the "scenery_packs.ini" file in your Custom Scenery directory
6. The new Links are now on Top of the other Scenery in that document, cut it and paste it at the bottom of the document (or above custom meshes if you've them installed)
7. Save the file, start the Simulator and enjoy!
